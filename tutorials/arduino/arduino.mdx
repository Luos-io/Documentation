## 1. Introduction to Arduino

<p>
  <a href="https://www.arduino.cc/" target="_blank">
    Arduino<IconExternalLink width="10"></IconExternalLink>
  </a>{' '}
  is an open-source electronic platform based on easy-to-use hardware and software. Luos engine can
  be used in Arduino IDE and examples are provided to test it on Arduino hardware quickly. For now,
  only Arduino SAM Board 32-bits ARM cortex-M0+ are compatible with Luos engine (Arduino Zero,
  Arduino MKR Wifi, Arduino MKR FOX, Arduino MKR WAN, Arduino MKR GSM, Arduino NB, etc.).
</p>

Creating a Luos network with an Arduino board is very easy. See the [hardware topics](/docs/hardware-consideration) page for more information.

## 2. Let's started with Arduino IDE

1.  Install the Ardiuno IDE from the <a href="https://www.arduino.cc/" target="_blank">Arduino<IconExternalLink width="10"></IconExternalLink></a> website.
2.  Copy and paste the following URL into the _File > Preferences > "Additional Boards Manager"_ textbox.

```json
https://raw.githubusercontent.com/Luos-io/Arduino_core/main/package_luos_index.json
```

<div align="center">
  <Image src="/assets/images/arduino_board_luos_preferences.png" />
</div>

3.  Install the Luos adapted Arduino SAMD Library in _Boards > "Add board definition" > Search for "Luos" > Install "Luos adapted Arduino SAMD (32-bits ARM Cortex-M0+) Boards"_

4.  Install the official Arduino SAMD Library in _Boards > "Add board definition" > Search for "SAMD" > Install Arduino SAMD Library_:

<div align="center">
  <Image src="/assets/images/arduino_Luos_board.png" />
</div>

5.  Download _Luos_Arduino\_\*.\*.\*.zip_ containing examples <a href="https://github.com/Luos-io/luos_engine/releases" target="_blank">on GitHub<IconExternalLink width="10"></IconExternalLink></a>

6.  Include Luos engine library for Arduino to your Arduino IDE:

<div align="center">
  <Image src="/assets/images/arduino_include_library.png" />
</div>

7.  Use provided Luos example, then compile and upload it to your Arduino board:

<div align="center">
  <Image src="/assets/images/arduino_Luos_example.png" />
</div>
